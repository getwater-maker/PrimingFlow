📖 사용 설명서

🚀 빠른 시작 (3단계)

  1. Google 로그인 — 계정 선택 후 [로그인] 클릭, 브라우저에서 로그인 완료

  3. 생성 시작 — 출력 폴더 선택 후 [생성 시작] 클릭

👤 캐릭터 참조 이미지

인물이 여러 장면에 걸쳐 일관된 외형을 유지하게 합니다.

  • [+ 추가] 클릭 → 캐릭터 이미지 선택 (여러 장 가능)
  • 정면 상반신, 깨끗한 배경이 효과적
  • 2~3명 이내 추천
  • 생성 시작하면 Flow에 자동 업로드됨

📂 출력 결과

출력폴더/
├── images/ — 생성된 원본 이미지 (1376x768 JPG)
├── subtitles/ — SRT 자막 파일
├── 주제명.vrew — Vrew 프로젝트 (바로 편집 가능)
├── original_script.txt — 원본 대본
├── translations.txt — 번역 결과
└── prompts.txt — 실제 사용된 프롬프트

💡 팁

  • 대본의 첫 단락이 배경/장소를 설명하면 일관성이 높아집니다
  • "백설공주" 같은 캐릭터명은 자동으로 일반 묘사로 변환됩니다
  • 갤러리 탭에서 이전 생성 결과를 확인할 수 있습니다
  • 출력 폴더의 .vrew 파일을 Vrew에서 열면 바로 영상 편집 가능
  • 여러 Google 계정을 추가하여 번갈아 사용할 수 있습니다
