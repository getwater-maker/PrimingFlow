## 상세 설명

### 한국어

우토그록(Uto Grok)은 xAI의 grok.com에서 동영상 생성을 자동화하는 Chrome 확장프로그램입니다.

🎬 주요 기능
• 텍스트 → 비디오: 텍스트 프롬프트로 동영상을 일괄 생성
• 이미지 → 비디오: 이미지를 업로드하여 애니메이션 동영상으로 변환
• 프롬프트 대기열: 수십 개의 프롬프트를 큐에 넣고 자동 처리
• 자동 다운로드: 생성 완료 시 자동으로 파일 저장
• 실패 자동 재시도: 설정한 횟수만큼 자동 재시도

⚙️ 풍부한 설정
• 기본 모드, 화면 비율, 비디오 길이, 다운로드 품질 설정
• 랜덤 딜레이로 rate limit 방지
• 프로젝트별 폴더 자동 정리
• 다크/라이트 테마 전환
• 설정 내보내기/가져오기 (JSON)
• 한국어, English, 日本語, 中文 지원

⌨️ 편의 기능
• Ctrl+Enter: 실행
• Esc: 중지
• Ctrl+P: 일시정지/재개
• 예상 소요시간 실시간 표시
• 생성 히스토리 자동 저장

✅ 개인정보 보호
• 어떠한 데이터도 외부 서버로 전송하지 않습니다
• 모든 데이터는 사용자의 브라우저에만 저장됩니다

### English

Uto Grok is a Chrome extension that automates video generation on xAI's grok.com.

🎬 Key Features
• Text → Video: Batch generate videos from text prompts
• Image → Video: Upload images and convert them into animated videos
• Prompt Queue: Queue dozens of prompts and process them automatically
• Auto Download: Automatically save files when generation is complete
• Auto Retry: Automatically retry on failure up to configured attempts

⚙️ Rich Settings
• Default mode, aspect ratio, video duration, download quality
• Random delay to prevent rate limits
• Auto-organize downloads into project folders
• Dark/Light theme toggle
• Export/Import settings (JSON)
• Korean, English, Japanese, Chinese support

⌨️ Shortcuts
• Ctrl+Enter: Run
• Esc: Stop
• Ctrl+P: Pause/Resume
• Real-time ETA display
• Auto-saved generation history

✅ Privacy
• No data is ever sent to external servers
• All data stays in your browser
